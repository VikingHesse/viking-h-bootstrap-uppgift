Bootstrap uppgift

  • Element

    Ett HTML element definieras med en start-tag, innehåll och en slut-tag. Ett exempel kan vara: <tag>Innehåll</tag>

  • HTML

    (HyperText Markup Language) utgör strukturen för alla internets webbsidor i form av rubriker, styckeindelning, att särskilja visst innehåll från annat och liknande. Det går också att infoga bilder med HTML. HTML är ett märkspråk och inte ett programmeringsspråk, eftersom HTML-kod endast har i uppgift att ge struktur. För att ett språk ska få kallas programmeringsspråk behöver det finnas möjlighet att utföra kommandon, vilket inte går med HTML. HTML används i samarbete med CSS, vilket bestämmer stilen på webbsidan.

  • ID

    ID attribut används för att specificera ett unikt HTML element och det måste vara unikt på en webbsida. Det finns ingen speciell regler för vilka element som får använda sig av id, men typiskt använder man id på större områden som exempelvis sidomenyer, behållare för artikeltext, sidhuvud, sidfot och så vidare.

  • Class

    Class fungerar i princip på exakt samma sätt som id men samma class kan användas till flera element. Class används för att specificera ett unikt HTML element, precis som ID. Men det är mer vanligt att använda när man vill bestämma css.

  • Meta

    Meta är taggar som används i head delen av ett html dokument och är till för att definiera din hemsida. Till exempel så används en beskrivande meta tag av sökmotorn för att visa ett beskrivning av din hemsida i sökresultaten. Det finns flera olika meta taggar med olika funktioner.

  • Responsive

    Responsive handlar om att använda HTML och CSS för att automatiskt ändra storlek på olika element och liknande, så att webbsidan ser bra ut på mobila enheter.

  • HTTP

    HTTP (Hyper Text Transfer Protocol) är ett protokoll som används för att ta emot HTML dokument och liknande. Det är grunden av all datatransföring på nätet.

  • HTTPS

    HTTPS (Hyper Text Transfer Protocol Secure) är samma som HTTP, men det är mer säkert eftersom det är krypterat och används för säker kommunikation över internet. Väldigt viktigt för hemsidor med känslig information, som banker.

  • Compressed data

    Compressed data är data som har omkodats för att ta upp mindre utrymme. Det är väldigt nödvändigt när man skapar hemsidor eftersom det annars kan ta väldigt lång tid att ladda in data.

  • CDN

    CDN (Content Delivery Network) är ett system som består att geografiskt placerade serverar som arbetar tillsammans för att förse snabbt internet, CDN gör det möjligt att snabbt överföra HTML dokument, bilder, CSS och mera från en server till din dator.

  • Local bootstrap

    Local bootstrap är när man har bootstrap (ett populärt HTML library) lokalt på en enhet istället för att länka till en annan server. Bootstrap finns att använda med hjälp av CDN, men ifall servern som har bootstrap på sig skulle vara nere så skulle inte bootstrap fungera på din hemsida, då kan det vara bra att ha det sparat lokalt.

  • JQuery

    Med JQuery kan man välja olika HTML element och utföra olika kommandon på dem. T.ex kan man använda $("p").hide() för att dölja alla <p> element.

  • Flexbox

    Flexbox (Flexible Box Module) är till för att enkelt kunna göra en layout av en hemsida, man kan enkelt skapa mellanrum mellan olika items och få en bra struktur på hemsidan.

  • Grid

    En CSS Grid layout är till för att dela upp hemsidan i flera olika delar, då man kan enkelt ha en del för t.ex. en sidomeny, en del för all text och en del för bilder.

  • Code snippets

    Code snippets är en liten del av kod som enkelt kan sättas in i ett HTML dokument, snippets används för att kunna implementera vanligt använd kod istället för att behöva skriva det flertal gånger.
