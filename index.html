Bootstrap uppgift

  • Element

    Ett HTML element definieras med en start-tag, innehåll och en slut-tag. Ett exempel kan vara: <tag>Innehåll</tag>

  • HTML

    (HyperText Markup Language) utgör strukturen för alla internets webbsidor i form av rubriker, styckeindelning, att särskilja visst innehåll från annat och liknande. Det går också att infoga bilder med HTML. HTML är ett märkspråk och inte ett programmeringsspråk, eftersom HTML-kod endast har i uppgift att ge struktur. För att ett språk ska få kallas programmeringsspråk behöver det finnas möjlighet att utföra kommandon, vilket inte går med HTML.

  • ID

    ID attribut används för att specificera ett unikt HTML element och det måste vara unikt på en webbsida. Det finns ingen speciell regler för vilka element som får använda sig av id, men typiskt använder man id på större områden som exempelvis sidomenyer, behållare för artikeltext, sidhuvud, sidfot och så vidare.
